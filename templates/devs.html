Trigger Happy

How to get involved ?

the source code of the project is available on GitHub, where you could fork it, make your improvments, create new services (with django module) like Twitter, FourSquare, Reddit etc... to expand the project

If you find some issues linked to the application, do not hesitate to submit a ticket and provide the maximum of details

Existing Modules

the following services could be helpful to start your own module

  • Evernote
  • GitHub
  • Pocket
  • Readability
  • RSS
  • Trello
  • Twitter
  • Wallabag

A skelton module intended to be cloned to make your own Trigger Happy module named django-th-dummy, permits, with very few new lines of code to produce his little module home made with Trigger Happy (link in french), or if you prefer, you can use an even quicker way to bootstrap your new Trigger Happy module with TriggerHappy Ansible

Comment participer ?

Le code source du projet est disponible sur GitHub, d'où vous pourrez le forker, pour apporter vos améliorations, créer de nouveaux services (via des modules Django) tels Twitter, FourSquare, Reddit etc... pour étoffer le projet

Si vous trouvez des problèmes liés à l'application, n'hésitez pas à ouvrir des tickets en les détaillant le plus possible
